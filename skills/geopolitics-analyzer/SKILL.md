# Geopolitics Analyzer

This skill provides geopolitical and macroeconomic trend analysis.

## Usage

Run the analysis script to get the latest trends:

```bash
node scripts/analyze.js
```

The script outputs a JSON containing current trends across different sectors. Use this data to formulate a comprehensive, logically sound, and slightly skeptical investment recommendation.
